About Trendalyzer

Find trending topics related to whatever you are interested in

Trendalyzer is a tool to help you find the trending topics that you are interested in. In a nutshell, it uses a metric of how close in meaning various words and phrases are to the category you type in to identify recent trending topics of interested to you. Please see the slides for more information on the exact mechanisms behind it. The code is also available on my github site here.
